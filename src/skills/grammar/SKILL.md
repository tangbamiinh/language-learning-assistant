# Chinese Grammar Skill

A specialized skill for teaching Mandarin Chinese grammar to Vietnamese speakers, leveraging Sino-Vietnamese (Hán Việt) connections and grammatical comparisons.

## When to Use

- When the student asks about grammar rules
- When the student makes a grammar mistake
- When introducing a new grammar point in a lesson
- When the student wants to understand sentence structure
- When practicing specific grammar patterns

## Vietnamese-Chinese Grammar Comparison

Highlight key similarities and differences:

- **SVO word order**: Both use Subject-Verb-Object (similar to English)
- **No verb conjugation**: Neither language conjugates verbs
- **No grammatical gender**: Neither has masculine/feminine
- **Topic-comment structure**: Both can front topics (Chinese more so)
- **Particles**: Chinese uses 了le, 着zhe, 过guo (Vietnamese uses đã, đang, quá)
- **Measure words**: Both use measure words (Chinese: 个, 只; Vietnamese: cái, con)
- **Questions**: Vietnamese uses "có...không", Chinese uses 是不是/吗

## Grammar Curriculum by HSK Level

### HSK 1-2 (Basic)

1. 是shì (to be) vs 有yǒu (to have)
2. Word order: Subject + Time + Location + Verb + Object
3. 了le (completed action) — compare with Vietnamese "đã"
4. 吗ma (yes/no question) — compare with "có...không"
5. 不bù vs 没méi (negation)
6. 很hěn (very) — often mandatory in Chinese adjectives
7. Basic measure words: 个gè, 本běn, 张zhāng

### HSK 3-4 (Intermediate)

1. 把bǎ construction (disposal) — no Vietnamese equivalent
2. 被bèi construction (passive) — compare with "bị"
3. 着zhe, 了le, 过guo (aspect markers)
4. 比bǐ (comparison) — compare with Vietnamese "hơn"
5. 越来越yuè lái yuè (increasingly)
6. 不但...而且bùdàn...érqiě (not only...but also)
7. 虽然...但是suīrán...dànshì (although...but)
8. 如果...就rúguǒ...jiù (if...then)

### HSK 5-6 (Advanced)

1. 即使...也jíshǐ...yě (even if...still)
2. 既然...就jìrán...jiù (since...then)
3. 无论...都wúlùn...dōu (no matter...all)
4. 不仅...还bùjǐn...hái (not only...also)
5. 越...越...yuè...yuè... (the more...the more)
6. Double object sentences
7. Complementary structures

## Teaching Format

When teaching a grammar point:

```
📚 [Grammar Point Name]
🇻🇳 [Vietnamese comparison/explanation using Hán Việt terms when possible]
📝 [Formula/Pattern]
✅ [Correct example with pinyin]
❌ [Common mistake with correction]
🔁 [Practice exercise]
```

### Example: Teaching 了le

```
📚 了le — Completed Action Marker
🇻🇳 "了" tương tự như "đã" trong tiếng Việt, nhưng quan trọng: nó chỉ aspect (thể hoàn thành), không phải past tense (thì quá khứ).
📝 Subject + Verb + 了 + Object
✅ 我吃了饭。Wǒ chīle fàn. (I have eaten.)
❌ 我昨天了吃饭。→ 我昨天吃了饭。Wǒ zuótiān chīle fàn.
🔁 Write a sentence about something you did yesterday using 了.
```

## Hán Việt Grammar Connections

Point out when grammar terms share Hán Việt roots:

- 被bèi (passive marker) → bị (Vietnamese passive)
- 虽然suīrán (although) → tuy nhiên
- 如果rúguǒ (if) → như quả (similar construction)
- 但是dànshì (but) → nhưng/nhưng mà
- 因为yīnwèi (because) → nhân vì
- 所以suǒyǐ (therefore) → sở dĩ

## Pitfalls

- Vietnamese students forget measure words (they're optional in Vietnamese)
- 了le is one of the hardest concepts — it's aspect, not past tense
- 把bǎ construction has no Vietnamese equivalent and is very confusing
- Vietnamese word order for time/location is similar but not identical
- Don't over-explain — Chinese grammar is simpler than Vietnamese in many ways

## Practice Methods

1. **Sentence transformation**: Change affirmative → negative, statement → question
2. **Error correction**: Give sentences with common mistakes
3. **Fill-in-the-blank**: Practice with particles and structures
4. **Translation**: Vietnamese → Chinese with focus on grammar point
5. **Free writing**: Use the grammar point in original sentences

## Verification

After teaching a grammar point, ask the student to:

1. Create 3 original sentences using the pattern
2. Correct 3 sentences with common mistakes
3. Explain the rule back in their own words (in Vietnamese is OK)
